1. Importe o install.sql
  2. Configure /application/config/database.php
  3. Configure /application/config/ion_auth.php
    1. Edite $config['site_title'] com o nome do seu projeto
    2. Edite $config['admin_email'] com seu email

    Se seu sistema não envia emails (exemplo: localhost) desabilite a função de ativar a conta por email:

    Edite $config['email_activation'] = TRUE; to FALSE

  4. 4) Não precisa editar o config.php
  5. Acesse http://seusite.com.br/login
  6. Faça login com os dados: admin@admin.com / password
  7. Vá para Usuários -> Ver Usuários
  8. Edite o usuário Admin de acordo com seus dados.
  9. Delete install.sql e install.html

Pronto, seu sistema está configurado!
